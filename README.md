# Stock_Market_Analysis

Data Analytics project analyzing stock market data using Excel, SQL, Power BI, and Tableau. Includes data cleaning, exploratory analysis, KPI calculations, and interactive visualizations for informed investment insights.

**Presentation link**

https://drive.google.com/file/d/1dNYxDZXmnaORbPb_RPL2EmwULJkgi8Pm/view?usp=drive_link

**Project Overview:**

This project focuses on analyzing stock market data to extract meaningful insights related to price movements, returns, trading volume, and overall market performance. The analysis was carried out using Excel, SQL, Power BI, and Tableau, following an end-to-end data analytics workflow.

The goal of this project is to demonstrate strong analytical skills, data handling capabilities, and the ability to transform raw financial data into actionable insights through visual storytelling.

**Project Objectives:**

Analyze historical stock price data to identify trends and patterns

Calculate key financial metrics such as returns and volatility

Perform data cleaning and transformation for accurate analysis

Use SQL queries to extract and aggregate insights

Create interactive dashboards for effective visualization

Support data-driven decision-making for investment insights

**Tools & Technologies Used:**

Excel: Data cleaning, preprocessing, basic analysis

SQL: Data querying, filtering, aggregations, KPI calculations

Power BI: Interactive dashboards and business insights

Tableau: Advanced data visualization and trend analysis

**Key Insights:**

Identified periods of high volatility impacting stock performance

Observed long-term upward and downward price trends

Analyzed volume spikes corresponding to major price movements

Calculated return patterns useful for investment evaluation

**Conclusion:**

This project demonstrates the practical application of data analytics tools to real-world financial data. By combining Excel, SQL, Power BI, and Tableau, the project showcases an end-to-end analytical approach—from raw data to actionable insights—suitable for data analyst and business intelligence roles.

**Future Enhancements:**

Add more stocks for comparative analysis

Integrate Python for advanced statistical analysis

Automate data refresh and dashboard updates
